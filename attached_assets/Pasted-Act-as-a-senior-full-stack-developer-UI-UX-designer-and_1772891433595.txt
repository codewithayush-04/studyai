Act as a senior full-stack developer, UI/UX designer, and software architect.

Help me build a modern AI Study Assistant Web Application for students.

Tech Stack:
Frontend:
- React.js (Vite)
- Tailwind CSS
- React Router

Backend:
- Supabase (Authentication + PostgreSQL Database + Storage)

AI Integration:
- Google Gemini API for generating summaries, quizzes, and flashcards from study notes.

Your task is to design and generate a complete production-ready project with clean UI/UX and scalable architecture.

Application Features:

1. Authentication
- Implement Supabase authentication.
- Support Google login and email/password login.
- Maintain secure sessions.
- Protect private routes.

2. Dashboard
- Display recent study sessions.
- Show previously uploaded notes.
- Provide quick actions like:
  - Upload notes
  - Generate summary
  - Start quiz
  - Review flashcards

3. Notes Upload
- Users can paste or upload study notes.
- Store notes in Supabase database.
- Each note should belong to a specific user.

4. AI Summary
- Send note content to Gemini API.
- Generate:
  - Summary
  - Key points
- Save generated results in database.

5. Quiz Generator
- Generate 5–10 quiz questions from notes.
- Include correct answers.
- Allow users to practice quizzes.

6. Flashcards
- Generate flashcards automatically.
- Question on front, answer on back.
- Allow flipping flashcards.

7. Study History
- Users can view previous notes.
- Show summaries, quizzes, and flashcards for each note.

8. UI/UX Design Requirements
Design a modern and clean interface with:

- Responsive mobile-first design
- Sidebar navigation
- Top navigation bar
- Dashboard cards
- Clean typography
- Student-friendly colors
- Smooth hover and transition effects
- Loading states and empty states

Pages to design:

- Login Page

- Summary Page

- Flashcards Page

src/

10. Services Layer

Create service files for:
- supabaseClient.js
- authService.js

11. Database Design

Create Supabase tables:

- users
- notes
- summaries
- quizzes
- flashcards
- study_sessions
