# 📚 AI Study App

An **AI-powered study assistant** designed to help students learn faster and more efficiently.  
This application allows users to **ask questions, generate smart notes, and improve their understanding using artificial intelligence.**

---

## 🚀 Features

- 🤖 AI-powered question answering
- 📝 Smart notes generation
- 📖 Study assistance for different subjects
- ⚡ Fast and simple user interface
- 🌐 Accessible from anywhere

---

## 🛠 Tech Stack 

**Frontend**
- HTML
- CSS
- JavaScript
- React.js

**Backend**
- Node.js
- Express.js

**Database**
- Supabase / MongoDB

**Other**
- AI API Integration
- Hosting: Replit

## 🔧 Installation

Clone the repository

git clone https://github.com/codewithayush-04/ai-study-app.git

Go to the project folder

cd ai-study-app

Install dependencies

npm install

Run the project

npm start
